# 리액트와 JPA를 사용하여 간단한 CRUD 프로그램 만들어보기.

## 사용 언어
+ Java Version 11
+ React, REACT Hook 방식
+ MySql 8
+ Spring Data JPA
+ Junit5

## 기능

1. 할일 등록
1. 할일 삭제
1. 할일 수정
1. 할일 리스트
